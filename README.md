# Kana Trainer

Eine Vue 3-App zum Üben der japanischen Schriftzeichen Hiragana und Katakana.

## Features

- Quiz-Modus für Hiragana, Katakana oder gemischt
- Level-System mit steigendem Schwierigkeitsgrad
- Fortschrittsanzeige und Statistiken

## Tech Stack

- [Vue 3](https://vuejs.org/) mit `<script setup>` SFCs
- [Vite](https://vitejs.dev/) als Build-Tool

## Entwicklung

```bash
# Abhängigkeiten installieren
npm install

# Entwicklungsserver starten
npm run dev

# Produktions-Build erstellen
npm run build
```
